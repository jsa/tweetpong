With Tweetpong you can easily embed tweets in blogs etc. It generates you a link to an image of a given tweet and you can use the link directly wherever you want. (No need to save the image somewhere else.) The service is open source and completely free to use. It runs on Google infrastructure so it can handle lots of load.

Enter tweet url:
| | | |

Grab link: http://tweetpong.appspot.com/23019320509.png

For feedback, bugs etc. tweet me.

Here's a bookmarklet Tweetpong for easy png'ing. Drag it to your bookmarks bar and click on a tweet page, it takes you here.

API is very simple:
http://tweetpong.appspot.com/<tweet id>.png
or, with custom width
http://tweetpong.appspot.com/<tweet id>-<width>.png
HTTP response 200 if ok, 304 or 500 on error.
